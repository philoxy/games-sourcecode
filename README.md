# games-sourcecode
source code for all my games!!

this includes games for...
* TI-83
* TI-84+CE
* EV3

which are written in...
* TI-BASIC (TI calculators)
* EV3Basic (Small Basic w/ extensions)

# how to install

## TI-83
* Install TI-Connect (not TI-Connect CE)
* Go to TI Data Editor -> TI Program
* Select the Desination Device as "TI-83"
* Paste in the code and save
* Go back to the home screen and click TI Device Explorer
* Connect your calculator to your computer
* Send the file to your calculator

## TI-84
* Install TI-Connect CE
* Go to the Variables tab (indicated by the 3 blocks on the left bar)
* Click "New Program"
* Paste in the code and save
* Go to the Connected Calculators tab (shown by the papers on the left bar)
* Connect your calculator to your computer
* Send the file to your calculator

## EV3
* Install EV3 Basic (windows only)
* Install Small Basic
* Make a new Small Basic file and paste in the code and save
* Open EV3 Explorer (should install with EV3 Basic)
* Connect your EV3 to your computer via the USB-B port
* Convert and send the file to your EV3

although, you could look at the source code directly from your calculator...
